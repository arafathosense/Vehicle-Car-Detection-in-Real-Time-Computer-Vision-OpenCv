# Vehicle Car Detection in Real Time (Computer Vision + OpenCV + Haar Cascade )
This project performs real-time vehicle (car) detection from a video using Computer Vision OpenCV and a Haar Cascade Classifier. It detects cars frame-by-frame and draws bounding boxes to highlight them visually. This is a great beginner-friendly Computer Vision project.
